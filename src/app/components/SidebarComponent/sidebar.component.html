Actions with 2d model

- translation
A (), D () - rotation
W, S - reflection X/Y
Space - reflection O
Q (-), E (+) - scaling
Z, X - stratching (horizontal/vertical)
C, V - compression (horizontal/vertical)
Click - set pivot

Info button - information about the current shape
Reset button - reset transformations and pivot

Actions with 3d model

- translation
W, S (y), A, D (x), Z, X (z) - rotation
C, V - scaling
Reset button - reset transformations
